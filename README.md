# 🌦️ Weather App

A minimalist and responsive web application that delivers real-time weather insights for any location.

## ✨ Key Features
* **Search Functionality:** Features an intuitive search input field allowing users to look up a location and instantly retrieve real-time data.
* **Weather Metrics:** Displays essential weather data including current temperature, general weather conditions (complete with visual icons), humidity levels, and wind speed.
* **Minimalist Design:** Built with a clean, modern user interface optimized for quick and accessible navigation.
